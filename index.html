Kaymak.js
Kaymak.js is a lightweight, flexible and multi-device content/image slider.
It's only 4kb gziped.

Plugin dependencies are jQuery, Hammer.js and optionally Transit.js (for css3 transitions)
Feel free to fork on Github

You can find demos and snippets on this page. If you want to now more open the source code.
It's completely free.
Browser support
Safari, Chorme and Firefox
also with little glitch works on IE9 and IE10 both desktop and mobile.

Config
data-full:
on/off page based navigation.
data-full: "1" | "0"
data-fixed:
if you use data-fixed-width/data-fixed-height options, you must set it to "1".
data-fixed: "1" | "0"
data-fixed-width:
use it when you need to set dimension in full mode.
"full" get window width, parent get parent width and you can give specific number like "500"
data-fixed-width: "full" | "parent" | "int"
data-fixed-height:
use it when you need to set dimension in full mode.
"full" get window height, parent get parent height and you can give specific number like "500"
data-fixed-height: "full" | "parent" | "int"
data-duration:
if you set, slider moves next block or page every x second
optional
data-duration: "int" / in millisecond
data-loop:
if you set "1", slider goes back to first block or page after the last block or page.
optional
data-loop: "1" | "0"
data-liquid:
if you want to use liquid effect set it to "1".
data-liquid: "1" | "0"
data-navigator:
#id of the navigation container
data-navigator: "#id"
data-navigatortype:
#id of the navigation container. If you want to use thumbnail,you must add url to .k-block as "data-thumbnail" and give width and height to .k-image > img as "data-width" and "data-height"
data-navigatortype: "bullet" | "number" | "thumbnail"
data-next:
#id of the navigation next button
data-next: "#id"
data-prev:
#id of the navigation previous button
data-next: "#id"
